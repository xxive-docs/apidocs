    Introduction

    API Key Setup

    • All endpoints will require an API Key/secret Key.
    • Never share your API Key/secret Key.
    If your API key is inadvertently exposed, delete it immediately and generate a new key.

    General Info

    General API information

    • Base API URL
      • https://api.xxive.io

    • Root Endpoint

      • /api/v1

    • All endpoints return either a JSON object or array.

    • Users can use the Open API provided by Xxive to develop the following functions for trading.

      • [Spot API]
        • Spot Wallet
        • Spot Market
        • Spot OrderBook

    Endpoint Security Type

    • API Key / Auth Key are passed into the Rest API via the x-api-key / Authorization header.

    Limit of requests

    • Limit rule : 8 times/1s (IP), 200 times/1m (IP)
    • In same IP, all APIs we provide are limited to 8 times per second and 200 times per minute to ensure smooth service.
